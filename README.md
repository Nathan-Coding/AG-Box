## AgentBox

Coding Test for AgentBox

## Requirements
<a href="https://www.php.net/downloads.php" target="_blank">PHP 7 or latest version</a>

## How to testing
- open terminal then into the project root path, run unittest.
```shell
$ ./vendor/bin/phpunit tests
```
